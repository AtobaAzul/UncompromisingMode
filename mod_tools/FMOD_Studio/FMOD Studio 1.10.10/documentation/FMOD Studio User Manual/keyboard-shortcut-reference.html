25. Keyboard Shortcut Reference

FMOD Studio features a number of keyboard shortcuts. These key combinations allow you to perform certain common actions without using a mouse or similar pointing device. Keyboard shortcuts allow you to save time by performing actions more quickly than would otherwise be possible.

A limited number of keyboard shortcuts are shown to the right of their associated commands in the "File", "Edit", "Create", "View", "Window", "Scripts", and "Help" menus. Keyboard shortcuts are not displayed in context menus other than text input field context menus.

This chapter lists all the keyboard shortcuts present in FMOD Studio, grouped by the windows and panes in which they function. Note that some keyboard shortcuts do different things depending on which window or pane is in focus. For example, "Ctrl+N" creates a new 3D event in the events browser, but creates a new VCA in the VCAs browser, and does nothing in the assets browser. Any keyboard shortcut that does different things depending on the window and pane in focus is listed more than once.

In addition to pure keyboard shortcuts, FMOD Studio also features a number of modifier keys that can be held down in order to do more when using a mouse or other pointing device.

Modifier Key Action
Ctrl Use fine control when dragging dials and slider handles.
Alt When dragging a dial, adjust that dial's random modulator, and add a random modulator if one is not already present.
